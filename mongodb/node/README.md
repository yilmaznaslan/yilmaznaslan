# Node Basics

- Create NestJS Project `nest new my-first-project`
- After creation `npm run start:dev`
- Install Mongoose `npm install @nestjs/mongoose mongoose`

Some NestJS Commands

- `nest new my-project` <- Creates a project
- `nest g users` or `nest generate users` <- Creates a module

Running and building

- `npm run start` (Start the app, compiled JS)
- `npm run start:dev` Staert the app in watch mode, auto-reload on save
- `npm run build` Compile TypeScrtip -> JavaScript
