# Welcome to My Blog

This is an example blog post to help you get started with writing blogs.

## How It Works

1. **Write your blog posts** in the `yilmaznaslan/blog/` directory as markdown files
2. **Add frontmatter** at the top with title, date, excerpt, and tags
3. **The portfolio site** automatically copies and displays them
4. **Your blogs** are available both on your website and in your public GitHub repo

## Features

- Markdown support with full formatting
- Automatic date sorting
- Tag system
- Clean, readable design
- SEO-friendly URLs

## Getting Started

Simply create a new `.md` file in the `yilmaznaslan/blog/` directory with the frontmatter format shown above, and start writing!
