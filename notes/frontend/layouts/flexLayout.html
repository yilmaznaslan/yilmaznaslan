Explanation

1. Flex Container (.flex-container): • display: flex; turns the container into a flexbox container. • flex-direction: row; lays out items in a horizontal row. • flex-wrap: wrap; allows items to wrap onto multiple lines if necessary. • justify-content: space-around; distributes items with space around them along the main axis. • align-items: center; aligns items in the center along the cross axis. • align-content: flex-start; aligns multiple lines of items at the start of the cross axis. 2. Flex Items (.flex-item): • flex-grow: 1; allows items to grow equally to fill the remaining space. • flex-shrink: 1; allows items to shrink if necessary to avoid overflow. • flex-basis: 150px; sets the initial size of each item before distributing extra space. • order: 2; changes the visual order of the items. • align-self: flex-end; allows specific items to override the alignment set by align-items.

Viewing the Effects

• Order: The first item has order: 1; and appears first despite being defined first. • Flex-grow: The first item grows more (using flex-grow: 2;) compared to others. • Flex-shrink: The second item has flex-shrink: 0; and will not shrink even when space is tight. • Align-self: The third item is aligned to the bottom of the container using align-self: flex-end;.
