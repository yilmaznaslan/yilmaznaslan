# Threads and Async Practice

This directory contains practice exercises based on the THREADS_AND_ASYNC_GUIDE.md.

## Suggested Practice Structure

Create packages for different concepts:
- `basics/` - Basic thread creation and lifecycle
- `threadpools/` - ExecutorService, ForkJoinPool examples
- `completablefuture/` - CompletableFuture examples
- `blocking/` - Blocking vs non-blocking examples
- `mdc/` - MDC context propagation examples
- `performance/` - Performance testing and benchmarking
